to_framework_dataset()[source]

Convert the pipeline to a framework-native dataset.

Subclasses implement this to return framework-specific datasets (e.g., tf.data.Dataset for TensorFlow, torch.utils.data.Dataset for PyTorch). Enables high-performance features like batching, prefetching, and distributed training.

Returns:

Framework-native dataset object.

Any
